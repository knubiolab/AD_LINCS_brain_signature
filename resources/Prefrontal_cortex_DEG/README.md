# Prefrontal cortex DEG files

Place DEG gene-list files here. Each file should contain one gene symbol per line.

Expected names are listed in `../deg_manifest.tsv`, for example:

```text
Astrocyte_up.txt
Astrocyte_down.txt
Excitatory_neuron_up.txt
Excitatory_neuron_down.txt
Inhibitory_neuron_up.txt
Inhibitory_neuron_down.txt
Microglia_up.txt
Microglia_down.txt
Oligocyte_up.txt
Oligocyte_down.txt
```
